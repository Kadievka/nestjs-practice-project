## Description

This is a practice project built in:

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

It's services are the providers for a backoffice App, to manage users.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
